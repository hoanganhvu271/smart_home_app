## API Reference
Backend Server: https://github.com/hoanganhvu271/iot-server.git
API Docs: https://iot-server-siz9.onrender.com/api-docs

#### Get data sensor

```http
  GET /api/get-data-sensor
```

#### Get device action history

```http
  GET /api/get-device-action
```

#### Turn on Led

```http
  POST /api/turn-on
```

#### Turn off Led

```http
  POST /api/turn-off
```
